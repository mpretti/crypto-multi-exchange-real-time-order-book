All action buttons now have predictable IDs:

  • reconnect-{exchangeId} - For disconnected/error exchanges
  • disconnect-{exchangeId} - For connected exchanges
  • details-{exchangeId} - For all exchanges
  • log-toggle-{exchangeId} - For individual exchange log toggles

‚úÖ Automation Helpers

Available in browser console as window.automationHelpers:

  • testReconnect('binance') - Test reconnect for specific exchange
  • testDisconnect('binance') - Test disconnect for specific exchange
  • toggleExchangeLogs('binance', true/false) - Toggle logs for specific exchange
  • getExchangeLogStatus('binance') - Get log status for specific exchange
  • getExchangeIds() - Get all available exchange IDs
  • getExchangeStatus('binance') - Get exchange status
  • listAutomationIds() - List all available automation IDs
  • clearSavedSettings() - Clear all saved settings from localStorage
  • exportSettings() - Export current settings for backup/debugging

‚úÖ Settings Persistence

All user preferences are now automatically saved:

  • Individual Exchange Logs - Per-exchange log toggle states
  • Global Log Pause - Pause/resume state for all logs
  • Filter Settings - Status, type, sort, and log level filters
  • Auto-restore - Settings automatically restored on page load

‚úÖ Reliable Targeting

Browser automation tools can now reliably target elements using:

  • Stable IDs that don't change with DOM updates
  • Predictable naming patterns
  • Helper functions for testing
